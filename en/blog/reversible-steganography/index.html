【学习笔记】基于差值拓展的可逆隐写方法

一、步骤概览

加密
  1. 将图像划分为多个像素对$(x,y)$，其中$x,y \in Z, 0 \le x,y \le 255$。
  2. 取像素对的均值$l$和差值$h$，具体如下。

$$ \left \{ \begin{aligned} &l = floor(\frac{x+y}{2}) \\ &h = x-y \end{aligned} \right. \tag{1} $$

  1. 对于待嵌入的秘密信息$m$，以差值拓展的方式进行嵌入，步骤为$h’ = h \times 2 + m$。
  2. 将得到的嵌入信息的差值$h’$通过(2)式进行计算，得到新的像素对$(x’,y’)$。

$$ \left \{ \begin{aligned} &x’=l+floor(\frac{h’+1}{2}) \\ &y’=l-floor(\frac{h’}{2}) \end{aligned} \right. $$

  1. 增加限制条件，保证数据不会溢出，可描述为以下条件。

$$ |2\times h + m| \le min(2\times(255-l), 2\times l+1) $$

解密
  1. 对于加密后的像素对$(x’,y’)$，根据(1)式计算差值$h’$和均值$l’$。
  2. 对于得到的差值$h’$取LSB，即为密文$m$。
  3. 计算$floor(\frac{h’}{2})$即为原有的差值$h$。
  4. 通过均值$l’$（这个值与原有均值一致）和还原出来的原有差值$h$，通过式(2)计算像素对$(x,y)$，即为原有的像素对。

二、例子

加密
  1. 选取图像像素对(126, 96)，则为$x=126, y=96$。
  2. 计算差值和均值

$$ \left \{ \begin{aligned} &l = floor(\frac{x+y}{2}) =floor(\frac{126+96}{2})=111\ &h = x-y = 126-96 = 30 \end{aligned} \right. \tag{1} $$

  1. 差值30的二进制表示为11110。
  2. 假设嵌入秘密信息$m=1$，将1加入到LSB上，即执行$h’ = h \times 2 + m$，结果为111101，十进制即为61。
  3. 将$h’=61$代入到(2)，得到加密后的像素对为$(142,81)$。

$$ \left \{ \begin{aligned} &x’=l+floor(\frac{h’+1}{2}) = 111+floor(\frac{61+1}{2}) = 142\\ &y’=l-floor(\frac{h’}{2}) = 111-floor(\frac{61}{2})=81 \end{aligned} \right. $$

解密
  1. 已知$(x’,y’)=(142,81)$。
  2. 计算$l’$和$h’$，过程如下：

$$ \left \{ \begin{aligned} &l’ = floor(\frac{x’+y’}{2}) =floor(\frac{142+81}{2})=111\ &h’ = x’-y’ = 142-81 = 61 \end{aligned} \right. \tag{1} $$

  1. 取密文$m=LSB(h’)=LSB(111101)=1$。
  2. 计算原有差值$h=floor(\frac{h’}{2})=floor(\frac{61}{2})=30$。
  3. 得到原有像素对，过程如下，最后还原出原有的像素对$(126,96)$。

$$ \left \{ \begin{aligned} &x=l’+floor(\frac{h+1}{2})=111+15=126 \\ &y=l’-floor(\frac{h}{2}) = 111-15 = 96 \end{aligned} \right. $$

三、代码

主要实现式子（1）和式子（2）。

对于式子（2），通过均值和差值计算对应的像素对，函数如下。

def convHL(h, l):
    if h >= 0:
        x = l + int((h + 1) / 2)
        y = l - int(h / 2)
    else:
        x = l + int(h / 2)
        y = l - int((h - 1) / 2)
    return x, y

对于嵌入过程，函数如下：

def embed_msg(img, msg):
    ori_img = img.copy()
    H, W = img.shape[:2]
    img_data = img.reshape(-1)
    loc_1 = []
    loc_2 = []
    lsb_2 = {}
    len_msg = len(msg)
    for i in range(len_msg):
        # 取像素对，相邻两个即可
        x = img_data[2 * i] * 1.0
        y = img_data[2 * i + 1] * 1.0
        # 计算均值和差值
        h = x - y
        l = int((x + y) / 2)
        # 正常情况下可进行差值拓展后的结果
        hs_1 = 2 * h + msg[i]
        # 若不可拓展则考虑直接覆盖LSB，则需要记录原有的LSB了
        hs_2 = 2 * int(h / 2) + msg[i]
        # 优先拓展
        if abs(hs_1) <= min(2 * (255 - l), 2 * l + 1):
            # 均值一直不变，使用拓展后的差值计算对应的像素对
            xs, ys = convHL(hs_1, l)
            loc_1.append(2 * i)
        elif abs(hs_2) <= min(2 * (255 - l), 2 * l + 1):
            xs, ys = convHL(hs_2, l)
            loc_2.append(2 * i)
            lsb_2[2 * i] = int(h) % 2  # 原来的LSB内容
        else:
            print("no_type")
        img_data[2 * i] = xs
        img_data[2 * i + 1] = ys
    new_img = img_data.reshape(H, W)
    return loc_1, loc_2, lsb_2, new_img, ori_img

对于解密过程，函数如下：

def extractMsg(loc_1, loc_2, lsb_2, img_emd):
    H, W = img_emd.shape[:2]
    img_data = img_emd.copy().reshape(-1)
    n = len(loc_1 + loc_2)
    p_ext = []
    for i in range(n):
        loc = 2 * i
        xs = img_data[2 * i] * 1.0
        ys = img_data[2 * i + 1] * 1.0
        hs = xs - ys
        # 均值不变
        ls = int((xs + ys) / 2)
        # 取出的信息
        b = int(hs % 2)
        # 属于可扩展情况
        if loc in loc_1:
            # 恢复真正差值
            hs = (hs - b) / 2  
            x, y = convHL(hs, ls)
        # 属于可改变，强行改变了LSB的情况
        elif loc in loc_2:
            # 恢复真正差值
            hs = (hs - b) + lsb_2[loc]  
            x, y = convHL(hs, ls)
        p_ext.append(b)
        img_data[2 * i] = x
        img_data[2 * i + 1] = y
    res = img_data.reshape(H, W)
    return p_ext, res
