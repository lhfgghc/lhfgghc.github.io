1. Python语言介绍

1.1 基本概念

Python语言是一种解释型、面向对象、动态数据类型的高级程序设计语言。

1.2 语言特点

优点：

1.优雅、简单、明确

2.强大的标准库

3.良好的可拓展性

4.免费、开源

缺点

1.运行速度慢

2.加密难

3.缩进规则

4.多线程灾难

1.3 异常

1.3.1 六种典型的异常

1.除零错误（ZeroDivisionError）：除数为0

2.名称错误（NameError）：变量使用前未进行申明或者初始化

3.类型错误（TypeError）：某些函数或者方法只适用于特定的数据类型，如果 对数据类型的操作不当，就会产生类型错误（如数值＋字符串操作）

4.数值错误（ValueError）：在输入类型正确的情况下，具体输入值错误（如对字符串进行float操作）

5.索引错误（IndexError）：超出序列长度的索引操作

6.属性错误（AttributeError）：方法或者属性不适用该对象

1.3.2 异常抛出

raise语句，类似java中的throw

1.3.3 异常捕获
try:
	#正常代码
except(Exception1,Exception2),e:
    #异常发生时的处理代码
else:
    #正确时执行
finally:
     #无论对错都执行

x,y=map(int,input().split())
try:
    ans=x/y
except ZeroDivisionError:
    print("Error")
else:
    print(ans)  #没有任何异常抛出并捕获时才执行这一语句块
finally:
    print("TEST!")
1.3.4 with语句
with open() as f:
	content = f.readlines()
#等价于
f=open()
content=f.readlines()
f.close()

2. Python语法基础

2.1 Python数据类型

2.1.1 数值类型

1.长整型在整数后面带L（python3没有了）

2.浮点科学计数法 1e9之类的

3.复数：a+bj 或者 complex(a,b)

2.1.2 字符串

1.切片 s[a:b]，取[a,b)的字符

2.字符串格式化，与C的语法规则类似

print("%s,%d"%('Hellowrold',123))
#format格式
print("{},{}".format('hellworld',123))

3.空值None，与其他任何数据比较返回恒为False。未指定返回值的函数会返回None。

4.join，s1.join(s2),s2为字符串列表，通过s1串将s2给串联起来，最后返回一个字符串。

5.字符串中的数据（字符）是不可修改的。可以通过新的字符串进行整体赋值。

6.find函数。s.find(s1,st,ed),从s[st,ed),中查找s1是否存在，不存在则返回-1。

7.大小写修改。调用title函数进行首字母大写。upper与lower进行整体大小写转换。

8.删去空格。rstrip与lstrip函数分别删除左右边的空格，strip删除两侧空格。

9.替换。s1=s.replace(k1,k2)，将字符串s中的所有k1变成k2，返回改变后的串，原串不动。

2.1.3 类型转换

1.eval(str)函数，计算str中有效表达式的结果并返回。

ans=eval("1+2")
ans#结果为3

2.chr(x) ASCII转字符 ord(x) 字符转ASCII，汉字为Unicode

3.bin(x)、oct(x)、hex(x) 分别为x转二、八、十六进制字符串，含前缀0b,0o,0x。

4.exec函数：将字符串中的有效语句进行运行。

2.2 变量与常量

1.同一个变量可以多次赋值，变量本身类型不固定，即为动态语言。

2.变量类似指针，指向对象实例。

3.当变量不再需要时，python会自动回收内存空间，也可以用del语句进行删除。

2.3 运算符与表达式

1.数值0被视为False，非0数值被视为True。空字符串、空字典、空集合、空列表也被视作False。

2.or为从左到右运算，返回第一个为真的值。0 or 3 or 0 or 4 结果为3。

3.位运算。与C一致。

4.成员运算符。 in和not in，判别某个成员是否在序列中。

2.4 序列数据结构

2.4.1 列表

1.删除列表元素。

del list1[x] 即删除list1列表中索引为x的元素。

list1.remove(x) 即删除其中值为x的元素。

list1.pop(x) 和del一致，当参数为空pop列表尾部。

2.索引取值。若为负数则从后边开始数，最后一个为-1，倒数第二个为-2，以此类推。

3.切片步长为负时。表示逆向切片。

a=[2,3,5,7,11,13]
b=a[-2:0:-1]  #结果为[11,7,5,3]  从-2（包括）切到0（不包括）

4.a[::-1]返回翻转序列

5.序列的复制

直接=赋值相当于两个变量指向了同一个序列。若要重新生成一个一样的序列，可以采用[:]全切片后赋值。

6.列表操作符。[1,2,3]+[4,5,6]结果为[1,2,3,4,5,6]; [1,2,3]*3结果为[1,2,3,1,2,3,1,2,3]

7.排序。list1.sort()直接排序原列表，返回None；sorted(list1)，不排序原列表，返回排序后的列表。

8.insert(tar,x)在指定的索引处插入x。

2.4.2 元组

1.元组生成。括号，逗号隔开，若只有一个元素，也要加逗号，如tup=(1,)。

2.元组不可修改，也不能用del删除个别元素。

3.操作符与列表类似，如重复、元组连接。

4.tuple(list)，将列表转化为元组。

5.多变量赋值，x,y,z=1,2,3.

2.4.3 字典

1.字典值可以是任何Python对象。大括号创建。

2.不允许同一个键出现2次，后一个出现的会覆盖前一个。

3.键必须不可变，因此不能用列表作为键，可以用数字、字符串、元组。

4.删除。可以用del删除整个字典；clear函数将字典清空。del dict[x]或者dict.pop(x)，将某一个键为x的键值对删除。

5.in操作，判断某一个键是否存在，与has_key函数作用相似，对value不适用。

6.keys函数返回键列表，values返回值列表，items返回键值对元组列表。

7.zip()函数。将两个列表(元组)进行组合，返回一个配对后的元组列表。键值互换：d1=dict(zip(d.values(),d.keys()))

8.可直接通过中括号进行键查值，也可以用get函数，若键不存在get函数返回None，而中括号会报错。

2.4.4 集合

1.通过大括号创建，与字典需区别。空集合需要用set()函数进行创建。直接{}创建的为空字典。

2.集合为无序不重复的序列。

3.-|&分别计算集合差集、并集、交集，^处理不同时存在于两集合中的元素。也可以用difference、union、intersection、symmetric_difference函数进行操作。

4.add添加，remove删除，

3.Python控制语句

3.1 列表生成式

1.基本格式: [f(x)for x in list_x if g(x)]，即遍历list_x，并将所有满足g(x)的元素x进行f(x)操作，生成一个新的列表。

list_x=list(range(30))
def f(x):
    return x*x
def g(x):
    return x%2==0
new_list=[f(x) for x in list_x if g(x)]

#相当于i为外层for，j为内层for
[(i,j)for i in range(4) for j in range(3)]
#相当于先进行左侧操作（i从0到2，每次j都从0到2）
[[i+j for i in range(3)] for j in range(3)]

2.字典的生成式：{f for k,v in dict.items()}

3.集合的生成式，和列表生成式一致。

4.enumerate函数。在遍历列表时返回元素索引和元素值。

for i,j in enumerate(['a','b','c'])：
	print i,j
#0 a
#1 b
#2 c

4.数据文件的读写操作

4.1 打开文件

1.open函数打开，返回文件对象

2.打开模式：第一个字母：r只读，w覆盖写，a追加；第二个字母：t文本类型，b二进制类型。

4.2 文件读写

1.read(size)函数，读取长度为size的字符串，若未给定或者为负则读取所有内容。

2.readline，读取整行，返回字符串。

3.readlines，读取所有行并返回字符串列表，此时并没有将字符串后的换行符去掉，可以用strip函数去掉换行符。

4.对于文件写操作，需要将打开文件的模式改为w\a，通过write函数进行写操作。writelines将一个列表中的所有数据一次性写入。

5.csv模块读写数据

path='X.csv'
f=open(path,'r')#以只读模式打开文件
reader=csv.reader(f)#生成reader对象
content=[]
for con in reader: #for循环遍历reader
    content.append(con)  #一次添加一行（条）
f.close()

f=open(path,'w')#以只写模式打开文件
writer=csv.writer(f)#生成writer对象
for con in content:
    writer.writerow(con) #一次写入一行（条）
f.close()

4.3 文件夹操作

1.os.getcwd()返回工作目录

2.os.makedir(path)创建文件夹，其中path中有的文件夹都会被创建

3.os.rmdir(path)删除文件夹，当且仅当目标文件夹为空。

4.os.listdir(path)返回给出下的所有文件与文件夹名。

5.os.path.dirname(path)返回路径名称，basename(path)返回目标文件名称，split(path)将路径和文件名分割返回元组。

5.Python函数与模块

5.1 lambda表达式

f=lambda x,y,z:x+y+z  #一般格式
print(f(1,2,3))
#将多个lambda表达式作为元素组成列表，即为转换表
F=[(lambda x:x**2),(lambda x:x**3),(lambda x:x**4)]
print(F[0](1),F[1](2),F[2](3))

lambda表达式可以作为函数返回值。

5.2 函数的定义与使用

1.参数传递：值传递。

def f1(x,K): #值传递，形参x,K指向原对象，即100与[1,1,1]
    x=2		#改变形参的指向，不影响原对象内容
    K=[2,3,4]
    return
def f2(x,K):
    x=2
    K[2]=3	#改变了原对象中的某一个元素的指向，因此原对象会发生改变，同理在其中用append操作也会发生改变。
    return
n=100
K=[1,1,1]
f1(n,K)
print(n,K)
f2(n,K)
print(n,K)

2.参数设定：

顺序传递：按照顺序

关键字传递：按照关键字，可以不按照顺序，但是在混合使用时，需要保证关键字传递后面没有位置参数了。

f(1,2,z=3,u=4)#正确
f(1,y=2,3,u=4)#错误

默认参数：必须放到参数列表的末尾。

不定长参数：例如def f(*args)：最终传入的参数会被打包为元组进入函数体内。且在传入该部分时若用关键词传入则会出错。

def pf(x,*args):
    print(x)
    print(args)
    return
k=[1,2,3]
pf(1,*k) #手动拆包，print的结果为(1,2,3)
pf(1,k) #print的结果为([1,2,3],)
pf(1,*k,k) #结果为(1,2,3,[1,2,3])

对于**，则会将多余的含有变量名的参数

def pf(x,**args):
    print(args)
    return
pf(2,a=1,b=2,c=3)  #结果为{'a': 1, 'b': 2, 'c': 3}

混合使用时，要按照顺序进行，如

def pf(x,*k1,**k2):
    print(x)
    print(k1)
    print(k2)
    return
pf(2,2,3,3,a=1,b=2,c=3)#此时k1为(2,2,3),k2为对应字典  #若pf(2,2,a=1,b=2,c=3,3,3)则会报错

5.3 all和any函数举例

all表示参数的列表中全为1时则返回1，反之返回0，any表示有一个1时结果则为1，反之为0

all([1 if n%k!=0 else 0 for k in range(2,n)])
#该段代码进行素数判定，首先通过列表生成式，生成一个01序列，当序列为全1时，表示全部的k都满足n%k!=0，因此为素数。

6.字符编码问题处理

1.中文字符集：GB2312 扩充：GBK， 标准编码集：Unicode

2.Python3有两种表示字符串的类型：bytes和str，str包含了unicode字符，使用encode函数可以将字符串转化为bytes，反之通过decode函数可以将bytes转化为unicode的str字符。

s="你好"
print(type(s),s)
s=s.encode()   #可以传参，如'gbk','utf-8'等参数
print(type(s),s)
s=s.decode()
print(type(s),s)
'''
结果如下：
<class 'str'> 你好
<class 'bytes'> b'\xe4\xbd\xa0\xe5\xa5\xbd'
<class 'str'> 你好
'''

3.sys.getdefaultencoding()函数可以获取python解释器的默认编码模式。

4.chardet包中的detect函数可以实现对编码方法的一个估计。

7.面向对象程序设计

7.1 类的名字空间

1.每个类有自己的名字空间，类名是空间名。

2.每个对象也有自己的名字空间，其中包含一些对象实例属性名，但其类中不包含。

3.存储在类名称空间中的名称是类的成员。

4.instance.method()等价于class.method(instance)，其中class时instance的类型。

5.可以通过isinstance来判断某个对象是不是某个类的实例。

7.2 类的定义

7.2.1 成员函数相关（self）

1.对象本身作为成员函数的第一个参数进行传递，即self。

2.类的成员函数必须包含一个参数self，且位于参数列表的第一位。

3.在类的成员函数体中访问其它类成员时需要以self作为前缀。

4.在类外通过对象访问成员时，不需要传递self。

5.在类外通过类名调用成员函数时，需要显式为self的参数传递。

7.2.2 构造函数与析构函数
class A:
	def __init__(self,sa,sb):#若构造函数中存在self以外的参数，那么在创建实例时需要传入具体的参数值。
        self.a=sa
        self.b=sb
    def __del__(self): #析构函数
        pass
7.2.3 实例属性与类属性

1.实例属性属于对象实例，是在构造函数中被定义的，定义时以self作为前缀，调用时只能通过对象进行调用。类属性是在方法之外被定义的，可以通过类或者实例对象进行调用，且被所有类对象实例所共享。

2.python可以动态的为类和对象增加成员。见书本P122。

7.2.4 公有、私有成员
class A:
    def __init__(self,sa,sb):
        self.a=sa	#公有成员变量
        self.__b=sb #私有成员变量   只能在类间调用，但可以通过：实例名._类名+私有成员名 的方法在类外进行访问。
    def __Q1():
        pass #属于私有的成员方法
    def Q2():
        pass #属于公有的成员方法

1.对于类的方法，可分为公有方法、私有方法、静态方法。具体示例见P125。

2.静态方法可以通过类或者对象进行访问，但在方法体中只能访问属于类的类属性，而不能具体地访问到某一个对象的属性。

7.3 类的继承和多态

7.3.1 类的继承

1.在继承中的基类的构造函数__init__()不会被直接调用，需要在派生类的构造函数中亲自专门调用。

2.在派生类中调用基类方法的两种形式。“基类名.方法名(self,)”和super语句。

class A(object):
	def __init__(self,x):
        self.AX=x
    def showA(self):
        print(self.AX)
class B(A):
    def __init__(self,x,y):
        A.__init__(self,x)
        #super(A,self).__init__(x)  也行
        self.BY=y
    def showB(self):
        A.showA(self)
        #super(A,self).showA()   #从派生类中调用基类成员的两种方法

3.python支持类的多继承。

7.3.2 方法重写

1.当基类和派生类都存在相同的方法时，可以在派生类中对方法进行重写，即覆盖。这样通过派生类或者其对象实例访问该方法时，总能够访问到被覆盖后的方法。

7.3.3 多态

1.举例：一方法以基类作为形参，则可将该基类的所有派生类对象作为实参，并顺利地进行调用。

7.3.4 运算符重载

对于python来说，所有加减乘除的运算符（+,-…）都有函数方法与之对应，见书本P131，需要重载运算符，则需要重写对应的函数方法即可。

def A:
	def __init__ (self,a):
        self.a=a
    def __add__(self,other):
        return ....#自定义的加法操作

8.命名空间和作用域

1.对于同名变量，若不声明global，则函数内的被认定为局部变量

x=100
def f():
    #global x
    y=x
    x=0
    print(x)
    return 
f()
print(x)

上述代码会报错，因为x为局部变量，在第一个语句y=x中，x并未被声明，加上对x的声明即可，这里通过global声明x为全局的。
